Что это за проект?


Сайт был создан в рамках обучения на курсе Яндекс-Практикум: Бэкенд-разработчик на Python

Основные цели:
  • Разобраться в работе IDE Visual Studio Code
  • Улучшить понимание принципов ООП
  • Познакомиться с возможностями фреймворка Django
  • Получить представления о фронтенд: HTML и CSS
  • Сделать первый проект для своего портфолио
  • Сдать все вовремя и не вылететь с курса

  • Python 3.7
  • Django 2.2.16
  • VS Code
  • Bootstrap
  • Божья помощь

Заложенные возможности по ТЗ заказчика:

  • Страницы: главная, страница сообщества, профайл автора с постами, карточка поста
  • Создание и редактирование постов
  • Создание комментариев к посту
  • Работа с сообществами (создание через админ-панель и его опциональный выбор в посте)
  • Возможность подписок на авторов (лента подписки с переключателем на главную страницу)
  • Лента постов на главной странице
  • Добавление картинок к постам
  • Настроенная админ-панель
  • Работа с пользователями: регистрация новых, вход/выход, смена пароля и восстановление пароля

Дополнительно разработано помимо пожеланий заказчика из ЯП:

  • Личный кабинет с полями за рамками стандартной модели
  • Удаление постов
  • Удаление комментариев
  • Пересмотрена структура навигации, добавлен мини-раздел пользователя
  • Немного красоты: кнопки действий вместо ссылок, картинки к кнопкам, посты в виде карточек
